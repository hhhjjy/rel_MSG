【项目目标】
我要实现一个 query-based relational MSG 框架，整体思路如下：

1. 输入是同一场景的多视角图像。
2. 使用 backbone 对图像进行编码，得到 image-level features。
3. 使用 bbox / ROI 从图像特征中提取 region-level bbox features。
4. 对 bbox features 做跨视图 attention，得到 refined object memory。
5. 对 image/global features 做跨视图 attention，得到 refined image memory。
6. 使用固定数量的 learnable object queries 查询 refined object memory，输出：
   - object_node_feat
   - object_attn / assignment map
   - object_exist_logits
   - optional object_cls_logits
7. 使用另一组 learnable place queries 查询 refined image memory，并融合 object-aware memory / object nodes，输出：
   - place_node_feat
   - optional place_exist_logits
8. 使用两个 MLP edge heads 显式预测：
   - place-place edges (pp_logits)
   - place-object edges (po_logits)
9. 后续我还会继续扩展：
   - object query 与 GT object 的 matching
   - pair proposal network
   - relation decoder
所以请现在就为这些模块预留接口，但先不要实现复杂细节。
